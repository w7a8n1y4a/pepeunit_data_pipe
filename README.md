## Основные этапы развёртывания
0. Установите пакеты при помощи команды:
   ```bash
   make install
   ```
1. Настройте `.env` файл по образцу `.env_example` - [Подробнее о переменных окружения](https://pepeunit.com/deployment/env-variables.html#backend-data-pipe)
1. Запустите приложение командой:
   ```bash
   make run-dev
   ```
